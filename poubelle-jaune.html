La Poubelle Jaune - Guide Complet

Que mettre dans la poubelle jaune ?

La poubelle jaune est destinée aux emballages recyclables. Voici ce que vous pouvez y déposer :

  • Tous les emballages en plastique : bouteilles, flacons, pots de yaourt, barquettes, films plastiques
  • Les emballages métalliques : canettes, boîtes de conserve, aérosols, barquettes en aluminium
  • Les briques alimentaires (Tetra Pak) : briques de lait, de jus de fruit, de soupe
  • Les emballages en carton : boîtes de céréales, suremballages, cartons de pizza (propres)

Les bonnes pratiques

Pour un recyclage efficace, suivez ces conseils :

  • Videz bien vos emballages, mais inutile de les laver
  • N'emboîtez pas les emballages les uns dans les autres
  • Écrasez vos bouteilles dans le sens de la longueur
  • Laissez les bouchons sur les bouteilles en plastique

Les erreurs à éviter

Ne mettez pas dans la poubelle jaune :

  • Les objets en plastique qui ne sont pas des emballages (jouets, ustensiles...)
  • Les emballages souillés par des restes alimentaires
  • Le verre (à mettre dans le conteneur à verre)
  • Les textiles et chaussures
  • Les déchets électroniques

Pourquoi trier ?

Le tri des déchets dans la poubelle jaune permet :

  • De préserver les ressources naturelles
  • De réduire les émissions de CO2
  • D'économiser de l'énergie
  • De créer des emplois dans le secteur du recyclage
